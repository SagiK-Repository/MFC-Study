# 1. 구현사항 및 배운점
- `CBitmap`으로 비트맵 처리를 연습했다.
- `CPaintDC` / 메모리 DC 개념을 학습하고 `BitBlt`함수를 활용했다.
- 뒤집기 게임을 설계하고 활용하는 연습했다.
- 배열 활용한 알고리즘과 랜덤을 활용해 타드를 섞는 연습을 했다.
